# Zone-Based Workforce Scheduler

A web-based scheduling system for managing teams, workers, and resource allocation across different zones using a calendar interface.

## Features

* Zone → Team → Worker hierarchy
* Time-slot based scheduling (9 AM – 6 PM)
* Assign resources to workers per slot
* Workload tagging: **light / heavy**
* Interactive calendar view
* Clean UI for quick allocation and tracking

## Tech Stack

* React (Vite)
* FullCalendar
* CSS

## Usage

* Select a zone and team
* Choose a worker
* Click on a time slot in the calendar
* Assign a resource and workload
* View scheduled tasks visually

## Project Structure

```
src/
  components/
  pages/
  types/
  styles/
```

## Future Improvements

* Backend integration (API + database)
* Authentication & role-based access
* Drag-and-drop scheduling
* Analytics for workload distribution

---

A simple system designed to visualize and manage field operations efficiently.
